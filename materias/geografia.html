Exercícios sobre os fusos horários.

  • 90° + 90 ° = 180°
  • 180°+ 180° = 360°
  • 360°÷ 76° = 24h
  • 360°÷ 24h = 15°

Para calcular fuso horário devemos ter o conhecimento das seguindes fórmulas:

Fórmula para calcular o fuso horário e a hora

F 1 - F2 = h.

h + h = h

Fómula para calcular o grau atravé da hora.

h * 15° = grau

Fórmula para calcular o fuso

grau÷ 15°= F

Fórmula para calcular a distância angular

grau+grau=grau
